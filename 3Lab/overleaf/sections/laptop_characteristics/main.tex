\section{Kompiuterio charakteristika}
\label{sec:kompiuterio_charakteristika}

Šiame skyriuje pateikiama naudoto kompiuterio charakteristika.

\begin{itemize}
    \item CPU – \texttt{Intel Core i5-6200U @ 2.30 GHz}.
        \begin{itemize}
            \item $2$ Branduoliai.
            \item $4$ Gijos.
        \end{itemize}
    \item RAM – $16$ GB.
    \item OS:
        \begin{itemize}
            \item Host OS – \texttt{Windows 10}.
            \item Guest OS, ant kurios buvo paleidžiama programa – \texttt{WSL2 Ubuntu 24.04}.
        \end{itemize}
    \item GPU – nenaudota (nėra \texttt{Nvidia}).
\end{itemize}
